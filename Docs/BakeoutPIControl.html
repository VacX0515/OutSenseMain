3. 적응형 게인 스케일링

lagScale = observedThermalLag / 20.0   (클램프: 0.3 ~ 2.0)
Kp = 1.5 x sqrt(lagScale)
Kd = 3.0 / sqrt(lagScale)

SV 상한

upperLimit = min(effectiveMaxTemp, rampedTarget + max(observedThermalLag, 5) x 1.5)

열지연이 작은 시스템에서 CH1 SV가 과도하게 올라가는 것을 방지

SV 하한

lowerBound = max(rampedTarget + observedThermalLag x 0.85, 샘플온도 - currentLag x 0.3)

정상상태에서 CH1이 목표+열지연 이상을 유지하도록 보장

6. 홀드 PI (2단계)

newCh1SV = targetTemp + holdError x Kp + integralTerm + dTerm
항목승온 단계홀드 단계
감쇠storedEnergy > 2°C, 최대 5%/cyc없음
에너지보상-storedEnergy x 0.15 x proximity없음
적분 Ki양: Ki, 음: Ki x 1.5대칭 Ki
SV 감소 제한3단계3단계 (동일)

홀드는 이미 평형 상태 → 감쇠/보상이 오히려 언더슛을 유발하므로 제거
